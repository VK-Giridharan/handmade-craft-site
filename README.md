# Handmade Craft Gifts - Mini E-commerce Site

A 5-page responsive mini e-commerce website for a handmade chenille stem craft gift startup.

## Tech Stack

- HTML5
- CSS3 (separate file)
- Vanilla JavaScript (separate file)
- No backend
- Fully responsive
- Elegant pastel theme (Playfair Display + Poppins fonts)

## Project Structure

```
/handmade-craft-site
├── index.html         (Home)
├── products.html      (Shop)
├── cart.html          (Cart)
├── customize.html     (Custom Order)
├── contact.html       (Contact + Portfolio)
├── css/style.css
├── js/script.js
├── assets/
│   ├── logo.svg       (Placeholder - replace with logo.png if desired)
│   └── products/      (Add product images here)
└── README.md
```

## How to Run

1. Open `index.html` in a browser, or
2. Use a local server (e.g. `npx serve .` or VS Code Live Server)

## Features

- Product grid with color selector, lighting option (+₹100), quantity
- localStorage cart management
- Cart edit (color, lighting, quantity) and remove
- WhatsApp order placement (pre-filled message)
- Custom order form (WhatsApp submission)
- Contact page with portfolio gallery
- Responsive design with mobile hamburger menu
